## 结论

当前报告的算术计算基本正确，维度总分 `68 / 100` 与等级 `待改进` 匹配；多数维度也引用了转写时间戳和原文证据，具备质检报告雏形。

但从产品级“客观、严谨、可复核”的标准看，当前报告只能作为初稿，不能直接作为最终教师评价。主要问题是证据边界不够清楚、视觉证据未真正进入评分、片段课被按完整课堂评价、关注公平结论存在过度推断风险。

## 合理部分

1. 分数结构合理：10 个维度满分合计 100，当前维度得分合计 68，与总分一致。
2. 等级判断合理：68 分落在 50-75 区间，对应 `待改进`。
3. 证据链初步成立：多数维度包含原文引用、时间戳、加分点、扣分点。
4. 低分方向基本可解释：课堂片段中学生回答简短、教师讲解主导、追问不足，因此“教学方式方法”“关注公平”“课堂效果”低于内容类维度有逻辑基础。

## 主要问题

### P0：视觉证据未真正进入评分

数据目录中存在关键帧，但当前评分卡中 `仪表教态`、`语言表达及板书设计` 的 `source_model` 仍为 `text`，报告也写了“未收到关键帧截图”。

这会造成两个风险：

- 用户以为系统已经完整分析了视频画面，但实际视觉维度只是文本兜底。
- 板书、教态、环境、学生状态等本应依赖画面的维度无法客观成立。

处理原则：

- 报告必须显式展示“视觉证据状态”：已视觉评分 / 文本兜底 / 不可评。
- 视觉维度没有视觉模型结论时，不应写成确定性评价。

### P0：片段课不应直接等同完整课堂评价

当前视频是约 3 分 30 秒片段，报告按 100 分完整课堂进行评价。

这会造成两个风险：

- “没有总结”“互动少”“参与不足”等结论可能只是因为片段截取范围有限。
- 对教师整体授课能力的评价容易过度外推。

处理原则：

- 小于完整课时的视频应标注为“片段分析”。
- 片段分析只能输出“片段内表现”和“风险提示”，不能直接输出完整课堂终评。
- 总分应标注低置信度，或拆成“片段质量分 / 完整课评价待复核”。

### P0：关注公平存在过度推断

当前报告认为“课堂仅有 1 名学生，其他学生被忽视”，依据主要来自 ASR 说话人标签和开头一句“还都还没来”。

但转写中同时出现“其他同学都来”，且 ASR 说话人分离不等于真实学生人数。仅凭 `student_1` 标签，不能证明“其他学生被完全忽视”。

处理原则：

- 没有学生名单、上台/发言计数、视觉画面或可靠说话人识别时，应写为“互动对象识别不足，存在公平性风险”，而不是确定“其他学生被忽视”。
- 关注公平可以扣分，但必须降低结论确定性。

### P1：红线判断需要区分“未发现”和“无法判断”

当前报告对部分红线写“未发现”，但视觉未进入评分时，着装、环境、安全行为等只能说“文本中未发现，画面证据不足”。

处理原则：

- 红线项应有状态：通过 / 触发 / 无法判断 / 需人工复核。
- 无法判断不能等同通过。

### P1：当前缺少可信度与复核状态

评分卡没有记录证据覆盖率、ASR 置信度、视觉覆盖率、片段时长、是否人工复核。

处理原则：

- 增加 `evidence_confidence` 或 `review_status`。
- 报告页展示“可直接采信 / 建议复核 / 不可终评”。

### P1：默认一轮 LLM 评分稳定性不足

当前 `num_rounds = 1`，没有标准差、交叉校准或人工样本标定。

处理原则：

- 产品化默认至少支持 3 轮抽样评分或同标准复核。
- 对关键维度给出评分稳定性提示。

## 对当前报告的人工复核判定

建议保留：

- 总分 68 可作为“片段初评”的参考值。
- `待改进` 的方向基本合理。
- 内容类维度高于互动/效果类维度，方向合理。

建议修正：

- `关注公平 6/15`：可保留为低分，但结论应改为“基于当前片段，仅识别到单一学生有效互动，公平性证据不足且存在风险”，不要断言其他学生被完全忽视。
- `仪表教态 3/5`、`语言表达及板书设计 3/5`：应标注为“视觉未完成，文本兜底，不纳入最终结论或需人工复核”。
- `课堂效果 8/15`：对“目标达成”应加限制语，只能说明“片段内学生能跟随”，不能说明完整课堂目标达成。
- 红线：应改为“文本未触发红线，视觉红线待复核”。

## 下一步产品改造

1. 增加“证据状态层”：文本、视觉、ASR、片段完整性分别给出状态。
2. 增加“片段/完整课”模式：片段只出风险与观察，不出终评。
3. 重写关注公平判定：没有可靠学生识别时降级为风险提示。
4. 报告页增加“质检员复核结论”：采纳、调整、退回重跑。
5. 建立 5-10 节人工标注样本，用于校准提示词和评分边界。
