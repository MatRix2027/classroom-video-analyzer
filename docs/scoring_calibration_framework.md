# 课堂视频智能分析 — 评分校准框架 v1.0

> **版本历史**: v1.0 (2026-06-06) — 基于KS投诉课校准 + 波哥反馈
> **迭代原则**: 每次新增校准课例后更新版本号，保留历史校准记录

---

## 一、核心校准原则

### 1.1 投诉课校准原则（最重要）

投诉课存在**家长主观放大效应**，需遵循以下原则：

| 原则 | 说明 | 实操 |
|------|------|------|
| **不盲从投诉** | 家长的情绪反应 ≠ 教学客观质量，投诉课评分不应自动压低 | 投诉课的评分逻辑与普通课完全一致 |
| **不忽视信号** | 家长投诉的维度往往是真实短板的放大镜，需要更深挖 | 在投诉命中的维度增加子维度检测 |
| **交叉验证** | 用客观行为指标验证投诉是否成立 | 投诉说"不关注孩子" → 检查互动次数分布 |
| **偏差标注** | 报告中标注"本课为投诉课"，提醒阅读者注意偏差 | 输出报告增加 `is_complaint_class: true` 标记 |

### 1.2 数学课专项原则

数学课（含逻辑思维、博弈论等）的评分需额外考察**题目-方法匹配度**：

| 维度 | 数学课专项考察点 | 示例 |
|------|----------------|------|
| 知识传授 | 数学概念的本质是否讲透，还是只讲计算步骤 | 必胜策略课：是否讲清"为什么这个策略一定赢"，还是只说"按这个方法就能赢" |
| 教学逻辑 | 解题逻辑链是否完整，学生能否复述推理过程 | 树状图方法：是带着学生画完就结束，还是让学生自己选择画哪条分支 |
| 重点难点 | 数学难点是否通过多角度/多方法突破 | 三张牌必胜：只用树状图讲一遍，还是用举例+对比+反例多角度突破 |
| 教学灵活性 | 学生提出非预期解法时，教师的应对 | 学生说"我直接试出来的"：是肯定后引导，还是忽略直接回到既定方法 |
| 启发引导 | 提问是否指向数学思维，而非仅指向答案 | "答案是多少" vs "你是怎么想到的" |

### 1.3 评分偏差校准

根据KS投诉课暴露的问题，当前评分体系存在**系统性偏差**：

| 偏差类型 | 表现 | 根因 | 校准方案 |
|---------|------|------|---------|
| **完成度偏差** | "老师把步骤都讲完了" → 给高分 | 评分标准重"是否完成"轻"完成质量" | 增加"过程质量"子维度 |
| **积极性偏差** | LLM倾向给出中上评分，极少给差分 | LLM的sycophancy倾向 | Prompt中强调"严守标准，7分才是及格线" |
| **形式偏差** | 有板书/笔记/总结就给高分 | 形式≠效果 | 增加"实质有效性"校验 |
| **维度独立偏差** | 各维度评分相关性过高 | LLM形成"整体印象"后各维度趋同 | 强制各维度独立评分，互不影响 |

## 二、QC-v4 维度校准细则

> 当前默认评分标准为 QC-v4，以下逐维度补充校准细则

### 2.1 知识传授（10分，权重0.10）

**原标准要点**：内容丰富、阐述准确、完成教学参数

**校准补充**：

| 等级 | 原标准 | 校准后标准（增加子维度） |
|------|--------|------------------------|
| 优[9-10] | 内容丰富，信息量符合指南设计 | 内容丰富 **且** 能讲清数学本质（不是只讲步骤）；学生能用自己的话复述核心概念 |
| 良[7-9) | 内容完整，阐述准确 | 内容完整，但本质挖掘不足；学生能跟但理解深度不够 |
| 中[5-7) | 内容基本覆盖，讲解不够清晰 | 按指南讲全知识点但**无任何拓展**；学生只会套步骤不会变通 |
| 差[0-5) | 知识讲解不清 | 知识性错误；严重遗漏核心知识点 |

**数学课专项检测项**：
- [ ] 是否讲了"为什么"而不只是"怎么做"
- [ ] 学生是否能解释概念的本质（不只是重复教师原话）
- [ ] 拓展内容是否与核心概念关联（而非为了拓展而拓展）

**扣分信号**（基于ASR文本）：
- 教师连续5分钟以上单向讲授无提问 → 扣0.5-1分
- 学生回答"不知道"后教师直接给答案 → 扣0.5分
- 多次出现"就是这样的""记住就行了" → 扣1分

---

### 2.2 熟练程度（10分，权重0.10）

**原标准要点**：教材处理得当，讲授自如，能结合自己的理解

**校准补充**：

| 等级 | 原标准 | 校准后标准 |
|------|--------|-----------|
| 优[9-10] | 讲授自如，能结合自己的理解和思考 | 对所有题目滚瓜烂熟，能灵活串联知识点；**应对学生非预期回答时不出错** |
| 良[7-9) | 能灵活应对常见问题 | 对常规题型熟练，挑战题偶有卡顿但能自我恢复 |
| 中[5-7) | 主要依靠固定话术 | 照指南讲解无拓展；学生提问偏离指南时应对生涩 |
| 差[0-5) | 讲解基本照本宣科 | 基础题目也讲解不流畅；出现明显知识漏洞 |

**扣分信号**：
- 教师口误后无法自我纠正 → 扣1分
- 回避学生提出的超纲问题 → 扣0.5分
- 明显在念PPT/指南 → 扣1.5分

---

### 2.3 重点难点（10分，权重0.10）

**原标准要点**：明确重点和难点，通过不同方法强调

**校准补充**：

| 等级 | 原标准 | 校准后标准 |
|------|--------|-----------|
| 优[9-10] | 明确重点难点，不同方法强调 | 精准把握难点，**用多种角度/方法突破**（非单一方法重复）；难点时间分配科学 |
| 良[7-9) | 重难点突出但本质挖掘不足 | 能识别重难点，但突破方式单一；学生听懂了但换题不会 |
| 中[5-7) | 知识点讲全但重难点不突出 | 简单内容重点讲，难点一句带过；时间分配头重脚轻 |
| 差[0-5) | 没有重点，知识罗列 | 无重点意识；全部内容平均分配时间 |

**数学课专项检测项**：
- [ ] 难点是否用了≥2种方法/角度讲解（如：举例+反例+类比+操作）
- [ ] 难点讲解时间占比是否合理（难点应占课堂时间30%+）
- [ ] 学生在难点处的错误回答是否被用来深化讲解

---

### 2.4 教学方式方法（10分，权重0.10）

**原标准要点**：不断更新教学方式，根据学生进度调整，提供及时反馈

**校准补充**：

| 等级 | 原标准 | 校准后标准 |
|------|--------|-----------|
| 优[9-10] | 方式灵活多样，根据学生差异调整 | **方法-题目-学生三者匹配**；答错后有针对性的纠正而非笼统鼓励 |
| 良[7-9) | 能切换不同讲法 | 方法较多样，但复杂问题应变策略有限 |
| 中[5-7) | 依赖常规讲授法 | 方法单一（多为重复讲解）；学生理解偏差时调整方式机械 |
| 差[0-5) | 方法僵化 | 只有一种方法；答错后无纠正或纠正无效 |

**KS投诉课暴露的关键问题**：
> 家长投诉"非要按既定方法讲"——这是教学方式方法的"中"级表现，但当前系统给了9分"优"
> **校准：当学生提出自己的想法但教师未予接纳/引导时，本维度不得评为"优"**

**新增扣分信号**：
- 学生提出非预期解法但教师忽略/打断 → 扣1.5分（直接封顶良）
- 教师反复说"按我说的来""先别管那个" → 扣1分
- 全堂课只使用一种教学方法 → 不得超7分

---

### 2.5 教学逻辑（10分，权重0.10）⭐ 最需校准的维度

**原标准要点**：突出主线，脉络清晰，逻辑性强

**问题诊断**：KS投诉课家长最不满的是"教学逻辑僵化"，但AI给了9.5分"优"。原因：当前标准只看"逻辑是否清晰"（步骤完整=高分），忽略"逻辑是否灵活"。

**校准重构**：

| 子维度 | 权重 | 优 | 良 | 中 | 差 |
|--------|------|----|----|----|----|
| **主线清晰度** | 40% | 脉络清晰，环节衔接自然 | 主线清楚但衔接略显生硬 | 有主线但经常跑偏 | 无主线，东一句西一句 |
| **教学弹性** | 30% | 能根据学生反应调整推理路径 | 能在固定路径上做小幅调整 | 只会按既定路线走，**不接纳学生不同思路** | 遇到非预期情况完全无法应对 |
| **思维尊重** | 30% | 主动追问学生"你怎么想的"，肯定非预期思路 | 容许学生表达不同想法但不深入引导 | **打断/否定学生的不同想法**，强行拉回既定路径 | 压制学生思维，只允许一种解法 |

**评分公式**：教学逻辑 = 主线清晰度×40% + 教学弹性×30% + 思维尊重×30%

**KS课例校准验证**：
- 主线清晰度：9分（博弈论必胜策略，步骤确实清晰）
- 教学弹性：5分（按既定方法讲，未根据安安的沉默调整策略）
- 思维尊重：5分（"是家长又不在了吗？"= 在追问沉默原因而非尊重学生节奏）
- 校准后：9×0.4 + 5×0.3 + 5×0.3 = 3.6 + 1.5 + 1.5 = **6.6分**（原9.5分）

**扣分信号**：
- 教师说"先按我说的来"/"这个方法不重要" → 教学弹性扣分
- 学生提出不同想法但教师未予回应 → 思维尊重扣分
- 全程无学生自主探究环节 → 教学弹性封顶良

---

### 2.6 组织教学（10分，权重0.10）

**原标准要点**：传达课堂规则，合理规划时间，鼓励学生参与

**校准补充**：

| 等级 | 原标准 | 校准后标准 |
|------|--------|-----------|
| 优[9-10] | 课堂规则清晰，时间管理科学 | 规则清晰 **且** 执行一致（不双标）；突发事件处理得当 |
| 良[7-9) | 基本规则明确，时间管理合理 | 规则基本明确但执行偶有松懈 |
| 中[5-7) | 规则模糊或执行不一 | 课堂纪律松散但不影响教学 |
| 差[0-5) | 无规则，时间管理混乱 | 课堂秩序严重失控 |

**扣分信号**：
- 开课超过3分钟未进入教学 → 扣0.5分
- 下课拖延超过5分钟 → 扣0.5分
- 对不同学生执行纪律标准不一致 → 扣1分

---

### 2.7 仪表教态（5分，权重0.05）

**原标准要点**：着装规范，姿态自信，积极肢体语言

**校准补充**：

| 等级 | 原标准 | 校准后标准 |
|------|--------|-----------|
| 优[4-5] | 自信姿态，积极肢体语言 | 仪表规范，**语气温和坚定**（非命令式） |
| 良[3-4) | 仪表得体 | 仪表规范但语气偏硬或偏软 |
| 中[1-3) | 仪表一般 | 语气生硬/命令式；对学生的态度有明显温差 |
| 差[0-1) | 仪表不规范 | 明显不当言行/语气 |

**新增扣分信号**（来自KS投诉课）：
- 对沉默学生使用"是家长又不在了吗？"等施加压力的话语 → 扣1分
- 语气温冷/催促（"快点""别磨蹭"）→ 扣0.5分
- 对积极学生热情但对沉默学生冷漠 → 扣1分（温差效应）

---

### 2.8 语言表达及板书设计（5分，权重0.05）

**原标准要点**：准确生动流畅，课件板书布局合理

**校准补充**：

| 等级 | 原标准 | 校准后标准 |
|------|--------|-----------|
| 优[4-5] | 语言精准严谨又生动 | 语言生动 **且** 数学表达严谨（如"必然""可能"等用语准确） |
| 良[3-4) | 表达流畅有条理 | 流畅但数学语言偶有不准；板书清晰但不够结构化 |
| 中[1-3) | 语言表达清楚 | 以复述指南为主；板书基础 |
| 差[0-1) | 语言表达不清 | 口误频繁；板书混乱或缺失 |

**数学课专项**：
- [ ] 数学专业术语使用是否准确（"必然"vs"可能"vs"不一定"）
- [ ] 是否用学生的语言解释数学概念（而非只用成人术语）
- [ ] 板书/笔记是否帮助学生构建知识框架（而非简单抄写结论）

---

### 2.9 关注公平（15分，权重0.15）⭐ 权重最大的维度之一

**原标准要点**：每位学员至少2次有效单独互动，平等机会，建设性反馈

**校准补充**：

| 子维度 | 权重 | 优 | 良 | 中 | 差 |
|--------|------|----|----|----|----|
| **互动机会分布** | 40% | 每位学生≥2次有效互动；上台次数差距≤1 | 多数学生≥2次互动；差距≤2 | 部分学生0-1次互动；差距≥3 | 明显偏心；个别学生被完全忽视 |
| **反馈质量** | 30% | 对每位学生的反馈都是个性化的（针对其回答内容） | 反馈较具体但偶尔笼统 | 反馈以"很好""不错"等笼统词为主 | 无反馈或反馈无效 |
| **沉默学生关怀** | 30% | 主动设计让沉默学生参与的方式（如指定回答简单题） | 会点名沉默学生但方式较生硬 | **连续追问沉默学生造成压力** | 完全忽视沉默学生 |

**评分公式**：关注公平 = 互动分布×40% + 反馈质量×30% + 沉默关怀×30%

**KS课例校准验证**：
- 互动分布：6/15（荣娇果冻积极互动，安安大志参与少，差距≥3）
- 反馈质量：6/15（"你太棒了"类笼统表扬为主，缺少个性化反馈）
- 沉默关怀：4/15（连续追问安安"是家长又不在了吗"，造成压力）
- 校准后：(6×0.4 + 6×0.3 + 4×0.3) × 15/10 = **7.8分/15分**（原8.5分）

**新增扣分信号**：
- 同一学生连续3次以上被第一个/最后一个点名互动 → 扣2分
- 对沉默学生连续追问而非耐心等待 → 沉默关怀子维度封顶中
- 表扬用语95%以上为"很好""太棒了"等无差别词 → 反馈质量封顶良

---

### 2.10 课堂效果及整体印象（15分，权重0.15）⭐ 权重最大的维度之一

**原标准要点**：积极包容的课堂环境，清晰及时的学习指导，促进批判性思维

**校准补充**：

| 子维度 | 权重 | 优 | 良 | 中 | 差 |
|--------|------|----|----|----|----|
| **学习目标达成** | 35% | 所有学生达标且能迁移应用 | 多数学生达标 | 部分学生达标 | 多数学生未达标 |
| **课堂氛围** | 35% | 学生主动提问/探究，气氛活跃 | 学生被动但配合，气氛平和 | **气氛沉闷，学生被动跟随** | 课堂冷场，学生明显抗拒 |
| **思维培养** | 30% | 学生展示了批判性思维/问题解决能力 | 学生能按引导思考 | 学生只给出答案无思考过程 | 学生无思考表现 |

**评分公式**：课堂效果 = 目标达成×35% + 课堂氛围×35% + 思维培养×30%

**KS课例校准验证**：
- 目标达成：9/15（必胜策略讲完了，但安安/大志能否迁移存疑）
- 课堂氛围：5/15（家长说"气氛沉重、毫无趣味"——但需校准：投诉课偏激，客观看课堂确实偏沉闷但非"沉重"）
- 思维培养：6/15（有引导但多为封闭式，学生以跟答为主）
- 校准后：(9×0.35 + 5×0.35 + 6×0.30) × 15/10 = **6.75分/15分**（原9.0分）

**投诉课校准因子**：
- 家长说"气氛沉重" → 客观检验：ASR中教师单口时间占比、学生发言频率、沉默时段
- 家长说"毫无趣味" → 客观检验：教师是否使用了游戏/操作/趣味元素
- 如客观指标与投诉一致 → 课堂氛围按投诉方向评分
- 如客观指标与投诉不一致 → 在报告中注明"家长情绪偏激，但客观指标显示课堂氛围尚可"

## 三、总分校准

### 3.1 KS课例校准前后对比

| 维度 | 原评分 | 校准后 | 降分原因 |
|------|--------|--------|---------|
| 知识传授 | 9.5/10 | 8.5/10 | 缺乏本质挖掘（"为什么这个策略一定赢"讲得不够） |
| 熟练程度 | 9.5/10 | 9.0/10 | 能讲但应对沉默学生策略单一 |
| 重点难点 | 9.5/10 | 8.5/10 | 树状图只作为工具展示，未让学生自主选择画法 |
| 教学方式方法 | 9.0/10 | 7.0/10 | 学生有不同想法时未予接纳；方法-题目-学生匹配度不够 |
| 教学逻辑 | 9.5/10 | 6.6/10 | 主线清晰但弹性差、思维尊重不足（核心错位！） |

## 四、Prompt 校准要点

在 `quality_assessment.md` 中需要增加的约束：

### 4.1 通用校准约束

```
## 评分纪律（CRITICAL）

1. **7分是及格线**：良[7-9)意味着"达标但不出彩"，不是"好"
2. **优分必须稀缺**：优[9-10]意味着"可作为培训素材的标杆"，全堂课最多2-3个维度评优
3. **维度独立评分**：各维度评分互不影响，不要因为整体印象好就抬高低分维度
4. **扣分必须有证据**：每个扣分的scoring_points必须引用具体原文+时间戳
5. **加分不能只看形式**：有板书≠板书好，有互动≠互动有效，有总结≠总结到位
```

### 4.2 教学逻辑维度专项约束

```
## 教学逻辑维度评分规则（CRITICAL）

教学逻辑不仅看"逻辑是否清晰"，还必须看"逻辑是否灵活"：

- 主线清晰度（占40%）：环节衔接是否自然，推理链是否完整
- 教学弹性（占30%）：当学生的回答偏离预设路径时，教师能否灵活调整
  - 优：主动追问学生的思路，根据学生回答调整讲解方向
  - 中：只会按既定路线走，不接纳学生不同思路
  - 差：打断或否定学生的不同想法，强行拉回既定路径
- 思维尊重（占30%）：教师是否尊重学生的思维过程
  - 优：鼓励学生表达不同想法，即使错误也先肯定再引导
  - 中：容许表达但不深入引导
  - 差：打断/否定/忽视学生的不同想法

**判定规则**：
- 如果教师存在"打断学生思路"或"强行按既定方法讲"的行为 → 教学弹性+思维尊重合计不得超12分（满分20）
- 如果学生沉默后教师施压式追问（如"是家长不在吗？"）→ 思维尊重不得超5分（满分10）
```

### 4.3 关注公平维度专项约束

```
## 关注公平维度评分规则（CRITICAL）

关注公平需考察三个子维度：
- 互动机会分布（占40%）：每位学生的互动次数是否均衡
- 反馈质量（占30%）：对每位学生的反馈是否个性化
- 沉默学生关怀（占30%）：对沉默/内向学生的关怀方式

**判定规则**：
- 同一学生连续3次以上被第一个/最后一个互动 → 互动分布扣2分
- 对沉默学生连续追问而非耐心等待 → 沉默关怀封顶中
- 表扬用语90%以上为"很好""太棒了"等无差别词 → 反馈质量封顶良
- 如存在学生0-1次互动的情况 → 关注公平总分不得超9分（满分15）
```

### 4.4 数学课专项 Prompt 片段

```
## 数学课专项评估（当课程涉及数学/逻辑思维内容时启用）

对每个评分维度，额外检查：
1. **题目-方法匹配度**：教师选择的教学方法是否适合这道数学题的特点？
   - 例：博弈论必胜策略适合"操作+树状图+归纳"，不适合"直接告诉结论+反复练习"
2. **思维可见性**：学生的数学思维过程是否可见？
   - 优：学生能说出"我为什么这样想"
   - 良：学生能说出答案但说不清思路
   - 中：学生只说"会了"但无法解释
   - 差：学生沉默或直接猜答案
3. **迁移能力检测**：换一道类似题，学生是否能独立解决？
   - 如果课堂中无法直接检测，在课堂效果维度降低1分并注明"迁移能力未验证"
```

### 6.1 何时更新本文档

| 触发条件 | 更新内容 |
|---------|---------|
| 新增测试课例 | 追加校准课例库 + 更新校准因子 |

### 6.2 版本管理

- **v1.0** (2026-06-06)：初始版本，基于KS投诉课校准
- **v1.x**：每新增1个校准课例递增
- **v2.0**：当校准课例≥5个时，做一次系统性回顾，可能调整维度权重

### 6.3 校准验证标准

当校准课例库积累≥3个课时，验证以下指标：
- **方向一致率**：AI评分方向（优/良/中/差）与人工判定一致的比例 ≥ 80%
- **偏差幅度**：AI总分与人工判定总分差值 ≤ 10分
- **投诉课校准率**：投诉课AI评分不再出现"家长投诉最严重的维度得最高分"的错位
